aula 07 - estudo de CSS

a sigla CSS é cascading style sheets é uma linguagem utilizada para estilizar pagina web utilizando esta lingugem podemos definir:

  • cor de fundo, texto e lementos.
  • fonte e tamanho das letras
  • posicionamento de elementos dentro da pagina
  • efeitos visuais e anomações

o CSS funciona em conjunto com o HTML, enquanto o HTML diz o que parece, o CSS diz com deve aparecer.
